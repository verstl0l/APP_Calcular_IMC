## ## Contextualização Acadêmica e Justificativa

O **Índice de Massa Corporal (IMC)** é uma métrica internacional adotada pela Organização Mundial da Saúde (OMS) para avaliar o grau de subnutrição, peso saudável, sobrepeso e obesidade em populações humanas.

Do ponto de vista do desenvolvimento de software e da informática em saúde, este projeto serve como um estudo de caso clássico para a aplicação de conceitos fundamentais de programação, tais como:

* **Manipulação de Variáveis e Tipos de Dados:** Tratamento de dados numéricos contínuos (ponto flutuante).
* **Estruturas Condicionais:** Implementação de lógica de tomada de decisão para categorização dos resultados com base em intervalos predefinidos.
* **Validação de Entradas:** Garantia de que dados inconsistentes (como peso ou altura iguais a zero ou negativos) não quebrem o fluxo da aplicação.

---

## ## Fundamentação Teórica

O cálculo do IMC é baseado na relação matemática entre a massa corporal (em quilogramas) e a estatura do indivíduo (em metros), expressa pela seguinte fórmula:

$$IMC = \frac{peso}{altura^2}$$

### ### Tabela de Classificação Utilizada

Os resultados gerados pela aplicação são classificados de acordo com os padrões da OMS:

| Intervalo de IMC | Classificação |
| --- | --- |
| Abaixo de 18,5 | Abaixo do peso |
| 18,5 – 24,9 | Peso normal |
| 25,0 – 29,9 | Sobrepeso |
| 30,0 – 34,9 | Obesidade Grau I |
| 35,0 – 39,9 | Obesidade Grau II |
| Maior ou igual a 40,0 | Obesidade Grau III (Mórbida) |
